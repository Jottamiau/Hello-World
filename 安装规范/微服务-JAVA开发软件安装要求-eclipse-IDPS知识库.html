微服务-JAVA开发软件安装要求-eclipse
作者:卢进桦于 2019年02月25日 发布在分类 / 开发 下,并于 2019年08月05日 编辑

目前我们微服务的开发软件的要求主要是通过Eclipse来作为IDE使用。

对于Eclipse的版本不做要求限制，建议优先使用高版本。

Eclipse安装完成之后需要配置如下几个插件：


一丶安装或升级最新eclipse版本

1. 若已安装则点击菜单栏help，选择Check for updates。

2. 等待更新完成后重启eclipse。

3. 若未安装进入官网下载地址如下： https://www.eclipse.org/downloads/packages

4. 进入官网首页按下图操作

5. 点击下载安装包

6. 点击安装，选择Java版本

7.解压安装路径自行选择

8.选择菜单栏的help下的eclipse Marktplace


9.安装完成后，按照 https://jingyan.baidu.com/album/fdffd1f84a845af3e98ca186.html 进行优化（可选）

10. （新团员忽略）PS：之前已有旧版的eclipse可以不卸载，打开新版eclipse工作空间选择同一个，插件重新安装一遍即可。

二丶安装阿里巴巴代码规范插件


1. 点击菜单栏help，选择Install New Software。

2. 点击Add填写如下内容

3. 在Name填（alibaba）Location填（https://p3c.alibaba.com/plugin/eclipse/update）

4. 点击OK，勾选下图红色框内内容，点击Next直至完成。

5. 在测试类中点击鼠标右键菜单找到阿里编码规约扫描（快捷键Ctrl+Alt+Shift+J）。


三丶安装findbugs插件


1. 点击菜单栏help，选择Install New Software。

2. 点击Add填写如下内容

3. 在Name填（findBugs）Location填（http://findbugs.cs.umd.edu/eclipse）

4. 点击OK，勾选下图红色框内内容，点击Next直至完成。


四丶安装maven插件


1. 打开Maven官网(http://maven.apache.org/download.cgi) 下载Maven压缩包

2. 创建Maven存放目录

3. 将下载好的Maven解压到Maven3.6目录下

4. 打开计算机属性选择高级系统设置

5. 在系统变量新增变量，变量名MAVEN_HOME,变量值复制刚刚安装的Maven目录点击确定。

6. 在path变量后加上%MAVEN_HOME%/bin;

7. 测试是否安装Maven成功，打开CMD窗口输入mvn -v 显示如下图则成功（前提要安装好JDK）


8. 打开Maven安装目录下的conf目录下的settings配置文件


9. 在mirrors结点下添加如下配置（配置的私有仓库）


<mirror>

<id>nexus</id>

<mirrorOf>central</mirrorOf>

<name>internal nexus repository</name>

<url>http://172.168.2.55:9999/repository/maven-public/<url>

</mirror>

10. 在Maven安装同级目录下创建repository目录

11. 在settings配置文件中配置本地仓库存放目录

12. 具体目录根据自己创建的目录配置

13. 在eclipse选择菜单栏中的Window下的Preferences

14. 点击Directory选择刚刚我们Maven解压后存放的目录然后Finish


15. 选择User Settings 在第二步选择Browser选择Maven下的settings文件



16. 点击Update Settings 然后Apply and Close完成安装


五丶JDK安装配置（优先使用JDK1.8）


1. 在官网下载好1.8版本的JDK并安装好。 （https://www.oracle.com/technetwork/java/javase/downloads/index.html）

2. 选择菜单栏中的Window下的Preferences

3. 点击添加后选择Standard VM 然后Next


六丶安装lombox插件


1. 下载地址:https://projectlombok.org/downloads/lombok.jar

2. 运行下载的lombok.jar文件

3. 点击Specify location选择eclipse安装目录点击Install/update完成安装


七丶安装Yaml Editor插件并配置yaml文件默认打开方式


1. 选择菜单栏的help下的eclipse Marktplace

2.在Find输入yaml点击go搜索

3.安装成功后的文件效果


八丶代码规范配置类文件注解作者是谁，时间，以及描述。


1.选择eclipse菜单栏中的Window下的Preferences

2.点击添加代码风格（New Java files）  点击Edit


3.应用并保存，填写内容如下。

/**

* @author ${user}

* @date ${date} ${time}

* @Description

* @version 1.0

*/

5.点击OK Apply and Close  在创建新Java文件时会有如下配置

6.关键词列表：

@author 作者名

@date 日期

@version 版本标识

@parameter 参数及其意义

@since 最早使用该方法/类/接口的JDK版本

@return 返回值

@throws 异常类及抛出条件

@deprecated 引起不推荐使用的警告

@see reference 查看引用

@override 重写

更多Javadoc用法可以访问：百度百科、官方文档。
